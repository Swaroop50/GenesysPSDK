Platform SDK Samples for Java

1. COM AB sample

Sample demonstrates:

  • how to create and use IConfService through ConfServiceFactory;
  • how to Create/Read/Update/Delete object on the Configuration Server;
  • how to receive notifications from Configuration Server about changes in config objects, see EventsSubscriptionSamples.java;
  • how to enable log4j2 logging.
2. T-Server sample

GUI Sample demonstrates:

  • how to create and use TServerProtocol;
  • asynchronous channel API for communication with Genesys Servers, see AgentModel.java, most PSDK-related code is here;
  • handling of the unsolicited messages from Genesys Servers, see ReceivedMessageHandler;
  • handling of the channel Open and Close events;
  • how to attach asynchronous events to GUI thread, see SwingInvoker.java;
  • how to setup ADDP and update it's parameters at a runtime;
  • how to enable log4j2 logging.
3. Stat Server sample

GUI Sample demonstrates:

  • how to create and use StatServerProtocol;
  • how to open and close statistic;
  • handling of the incoming statistic notifications, see StatServerMessageHandler;
  • how to attach asynchronous events to GUI thread, see SwingInvoker.java;
  • how to configure new WarmStandby.
4. Stat Server Console sample

Sample demonstrates:

  • how to implement IAsyncInvoker to execute user's callbacks inside ThreadPool.
5. New WarmStandby sample

Sample demonstrates:

  • how to configure new WarmStandby;
  • different reconnection scenarios to multiple endpoints, switching between endpoint pools (clusters);
  • handling of the RegistrationException;
  • implementation of the ESP based Server, see EspServer.java;
  • implementation of the ESP based Client, see EspClient.java;
  • how to filter sensitive data in logs (for ex. passwords), see initLogger();
  • how to enable log4j2 logging.
6. FlexChat sample

Sample demonstrates:

  • how to login multiple clients on Chat Server using one communication channel;
  • how to join to chat;
  • how to send message to chat.
7. Application Template sample

Sample demonstrates helpers to parse application settings, retrieved from Management Framework:

  • helper method that creates preconfigured Endpoint;
  • helper method that creates Warm Standby configuration.
8. Message server subscription sample

Sample demonstrates how to work with Message Server:

  • how to establish connection with Message Server;
  • how to handle Message Server events (opened/closed);
  • how to handle Message Server messages;
  • how to send a message to Message Server;
  • how to subscribe for notifications;
  • how to unsubscribe from notifications.
9. Cluster protocol sample

Sample demonstrates how to work with cluster protocol:

  • how to create and configure a cluster protocol;
  • how to write and set cluster protocol channel listener.;
  • how to open and close a cluster protocol;
  • how to send messages to a cluster protocol;
  • how to receive messages from a cluster protocol;
  • how to make synchronous and asynchronous requests to a cluster protocol.
